# 웹접근성이란?

팀 버너스 리(Tim Berners-Lee)



>웹이란 '장애에 구애없이 모든 사람들이 손쉽게 정보 공유할 수 있는 공간'

>모든 사람이 정보통신 기기나 서비스를 손쉽게 활용할 수 있도록 만드는 것.

>예를 들자면 장애인과 노인들을 위해 - 리모콘, 전화, 자동문 등의 제품.
